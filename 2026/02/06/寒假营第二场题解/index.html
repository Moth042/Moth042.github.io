¶B.NCPC

题目大意是说总共$n$名选手进行比赛，每个选手有一个美味值$a_i$，每次会随机选两名选手进行比赛，如果美味值相等，则两个人都淘汰，否则美味值较低的淘汰，问对于$1\leq i \leq n$，是否存在一种比赛安排使得$i$获胜。

首先，如果一种美味值出现的次数为偶数，那么他们是可以互相抵消的。那么我们可以看最大值出现次数的奇偶性，如果是偶数，就意味着，除了自身之外，其他数字都可以被他们淘汰，所以此时除了他们本身之外，其它任何数都存在一种可以获胜的方案；如果最大值个数为奇数，那没有数字能淘汰掉最大值，最大值必然在最后会剩下至少一个，并且淘汰掉所有其它数字，所以此时除了最大值本身，其他数字均没有获胜的方案。

¶F.x?y?n!

题目要求构造两个不同的数$x$和$y$，在满足$\gcd(x,y)=n$的前提下使得$x\oplus y$的值最小。

因为$|x-y|\leq x \oplus y$，$\gcd(x,y)=n$且$x\neq y$，所以$x$和$y$是$n$的不同倍数。则$|x-y|$至少是$n$的$1$倍，即$n\leq |x-y| \leq x \oplus y$，所以此时$x \oplus y$的最小值为$n$。

那么接下来我们考虑如何构造：首先将$n$左移得到的数一定会是$n$的倍数，将这个数加上$n$（或者按位或）之后的结果也一定是$n$的倍数。假设$n$的二进制形式有$bit$位，那么我们将$n$左移$bit$位，再加上$n$（或者按位或），那么这两个数的最大公约数一定为$n$，并且它们的前$bit$位按位异或之后均为$0$，后$bit$位按位异或后得到$n$，此时满足了$x \oplus y=n$。

¶H.权值计算

题目定义一个数组的权值为其所有前缀中不同元素的个数的总和，求所有非空子数组的权值之和。

由于子数组有$n^2$个，所以我们可以直接考虑每个元素对子数组的贡献。对于$i$位置，包括$a_i$的子数组$[l,r]$，显然满足：$l \leq i \leq r$ 。并且只有当$a_i$在子数组中第一次出现时才会增加不同元素的个数，所以此时我们假设$a_i$上一次出现的位置为$j$，则当$j < l \leq i$时$a_i$都是新元素，此时左端点有$i-j$种选择。再考虑$a_i$对当前子数组的贡献，从以$i$为结尾的前缀开始，到以$r$为结尾的前缀，$a_i$都会产生贡献，所以一共产生了$r-i+1$次贡献。又因为$r\in[i,n]$，所以总共产生了$\frac{(1+n-i+1)(n-i+1)}{2}$次贡献（等差数列求和），考虑上左端点的选择个数，则每个$a_i$对答案的贡献为$(i-j) \frac{(1+n-i+1)(n-i+1)}{2}$。最后对每个$a_i$都进行这样的计算，再累加求和即可、

¶J.终于再见

题目将所有的点按度数分等级，要求出每个点到等级严格高于它的点的最短距离。

我们在图论中学过，所有点的度数之和等于边数的两倍，那么就意味着，在$m$条边的情况下，所有的点最多被分为$O(\sqrt{m})$个等级。那么我们可以枚举每个等级，将所有等级高于它的点进行多源$BFS$来找到最小值，并且只对当前等级的点更新即可。时间复杂度为$O(\sqrt{m}(n+m))$。
